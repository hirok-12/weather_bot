# README

自動天気予報LINEBOT

## 概要  
毎朝テレビで天気予報を確認するのが面倒なので、友達登録すると毎朝6時に当日の天気を自動送信してくれるLINEBOTです。

さらに、メッセージで"明日の天気は？"などとテキストすると外部APIから情報を取得してテキストで自動返信をしてくれます。
